The Problem

Convergence is not accuracy.

Current global solvers use branch-and-bound. At real-world scale, they don't finish. So you use local methods—gradient descent, interior point, SQP. They converge. They report success.

But they only find the nearest valley, not the deepest one. You're shipping suboptimal solutions and calling them optimal.

  • Linearizing non-linear constraints to make problems "tractable"
  • Running 100× with random restarts and taking the best
  • Accepting "good enough" because global is "impossible"
